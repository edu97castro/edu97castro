## 👨‍💻 Introducing Myself

My name is Eduardo Castro. I'm from Córdoba, Argentina. I have a degree in Physics, a Graduate Certificate in Data Science and Machine Learning and a strong background in programming, math, and statistics.

My Physics thesis was focused on the usage of Graph Neural Networks (GNNs) for discovering new materials for anode-free lithium batteries. This involved the development of a library to automate data creation and processing and another for the model implementation. This project is mainly laveraged by [PyMatGen](https://pymatgen.org/) and   [Pytorch](https://pytorch.org/) libraries. If you're interested in this work, check out its [GitHub repository](https://github.com/edu97castro/Li_Adsorption_ML)!
